# RevConnect
RevConnect is a Java-based, console-driven social media application that simulates core functionalities of modern social platforms. It supports multiple user roles including Personal Users, Content Creators, and Business Accounts, each with role-specific features.

The application allows users to register, authenticate, manage profiles, create and interact with posts through likes, comments, and shares, build networks via follows and connections, and receive in-app notifications. Creator and Business accounts additionally support enhanced profiles, promotional content, analytics, and audience engagement.

RevConnect follows a layered architecture using Java, JDBC, and MySQL, with clear separation between UI, Service, DAO, and Model layers. The design is modular and scalable, making it suitable for future extension into web or mobile applications.

Tech Stack: Java, JDBC, MySQL, Log4j, JUnit
Architecture: Layered / Modular
Type: Console-Based Application.
